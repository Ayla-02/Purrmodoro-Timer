# Purrmodoro-Timer
I like cats, I like productivity, so here is a cosy cat-themed timer. Focus sessions, task tags &amp; weekly stats, all in one lil' html file.

---

## Features

♡ **Pomodoro timer** with focus, short break, and long break modes — all durations configurable
♡ **Task tagging** - label sessions by category (work, study, creative, admin, health, other)
♡ **Session log** - today's completed sessions shown with tag, task name, and time
♡ **Weekly stats** - session count, total focus minutes, best day, current streak, bar chart by day, and breakdown by tag
♡ **Animated SVG cat** - tail wags while the timer runs
♡ **Browser notifications** - get alerted when a session completes (opt-in)
♡ **Sound** - a gentle chime on completion via Web Audio API
♡ **Persistent storage** - sessions and settings saved to `localStorage`

---

## Usage

No installation required. Just open the file.

---

## Tech

♡ Vanilla HTML, CSS, JavaScript
♡ `localStorage` for persistence
♡ Web Audio API for the completion chime
♡ SVG for the animated cat
♡ Notifications API for browser alerts

---

## Customising

All timer defaults are adjustable in the Settings tab in-app. To change tag categories or colours, edit the `TAG_COLORS` and `TAG_LABELS` objects near the top of the `<script>` block in `purrmodoro.html`.
